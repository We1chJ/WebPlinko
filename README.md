# WebPlinko 🎮

A web-based Plinko game built with Phaser.js and Next.js. **Educational project only - no real money involved.**

The true Plinko game experience with **REAL** physics (no hidden chance manipulation)

## ⚠️ Disclaimer
This is a demonstration project. No real money is involved. Do not gamble with real money.

## Tech Stack
- **Game Engine**: Phaser.js + Matter.js
- **Frontend**: React + Next.js
- **Styling**: Tailwind CSS
- **State**: Zustand
- **Deployment**: Vercel

## Getting Started

```bash
git clone https://github.com/We1chJ/WebPlinko.git
cd WebPlinko
npm install
npm run dev
```

Open http://localhost:3000 to play.

## How to Play
1. Set your bet amount
2. Click **Bet** to drop a ball
3. Ball falls through pegs and lands in a multiplier slot
4. Use **Auto** tab for automatic betting every 0.25 seconds
